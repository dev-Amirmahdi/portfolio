## 🎯 About The Project

This is a **personal portfolio website** showcasing my skills, projects, and experience as a **Frontend Developer**. The website is designed with a focus on:

- ✨ Smooth and engaging user experience
- 📱 Fully responsive design for all devices
- 🌙 Dark/Light mode toggle
- 🚀 High performance and optimized loading

---

## ✨ Features

| Section | Description |
|---------|-------------|
| 🏠 **Hero Section** | Introduction with CTA buttons and social links |
| 👤 **About Me** | Personal info and statistics (experience, projects, clients) |
| ⚡ **Advanced Skills** | Technologies I master professionally |
| 📚 **Learning & Exploring** | Technologies I'm currently learning |
| 🚀 **Featured Projects** | Sample projects with GitHub and demo links |
| 📧 **Contact Form** | Direct email communication (via Formspree) |
| 🌙 **Dark Mode** | Toggle between light and dark themes |
| 📱 **Mobile Menu** | Optimized navigation for mobile devices |
